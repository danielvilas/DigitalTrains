# LocoBuffer from LocoLinx32U4
This firmware is a copy from [https://github.com/mrrwa/LocoNet/tree/master/examples/LocoLinx32U4] but in platform.io

On this repo also exist a new board with the Loconet Interface for use with this firmware

Configure the device on JRMI as "Digitrax" "Loconet LocoBuffer II.

For the Piko SmartBox light, use Intellibox  or Intellibox II
